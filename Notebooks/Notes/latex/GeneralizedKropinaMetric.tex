    \hypertarget{summary}{%
\section*{Summary}\label{summary}}

    The geomorphic fundamental ``metric'' function \(\mathcal{F}^*\)
prescribes the distance traveled by an erosion front in unit time. It
takes the form (for the bedrock channel SPIM model) of power functions
of the surface-normal erosion slowness covector components
\(\mathbf{p} = [p_x, p_z]\) multiplied by a position dependent rate
variable \(\varphi(\mathbf{r})\). This kind of fundamental function is
called a generalized Kropina metric on \(\mathsf{T}^*\mathsf{M}\). This
canonical metric is one of the family of so-called
\((\alpha^*,\beta^*)\) Finsler (Cartan) metrics that, when defined on
the cotangent manifold indicated by the \(*\) annotation, combine a
contravariant Riemannian metric tensor \(a^{ij}\) with a vector field
\(b^i\) (the equivalent on the tangent manifold employs a 1-form \(b_i\)
along with a covariant metric tensor \(a_{ij}\)). In this geomorphic
case, the Riemannian tensor is just the Euclidean metric tensor scaled
by \(\sqrt{\varphi(\mathbf{r})}\), while the vector field is simply
\({\varphi(\mathbf{r})}\) applied to the horizontal unit vector.



    \hypertarget{fundamental-function}{%
\section*{Fundamental function}\label{fundamental-function}}

    The co-Finsler/Cartan fundamental function for the geomorphic (SPIM)
Hamiltonian, derived from first principles, is:

 
            
    
    $\displaystyle \mathcal{F}_* = p_{x}^{\eta} \left(p_{x}^{2} + p_{z}^{2}\right)^{\frac{1}{2} - \frac{\eta}{2}} \varphi{\left(\mathbf{r} \right)}$

    

    For a gradient-scaling exponent of \(\eta=\tfrac{1}{2}\) in the SPIM, we
have

 
            
    
    $\displaystyle \mathcal{F}_* = \sqrt{p_{x}} \sqrt[4]{p_{x}^{2} + p_{z}^{2}} \varphi{\left(\mathbf{r} \right)}$

    

    while for a gradient-scaling exponent of \(\eta=\tfrac{3}{2}\) in the
SPIM, we have

 
            
    
    $\displaystyle \mathcal{F}_* = \frac{p_{x}^{\frac{3}{2}} \varphi{\left(\mathbf{r} \right)}}{\sqrt[4]{p_{x}^{2} + p_{z}^{2}}}$

    

    \hypertarget{conversion-to-generalized-kropina-form}{%
\section*{Conversion to generalized Kropina
form}\label{conversion-to-generalized-kropina-form}}

 
            
    
    $\displaystyle \left( \alpha^*_{\mathrm{Kr}} = \sqrt{p_{x}^{2} + p_{z}^{2}} \varphi{\left(\mathbf{r} \right)}, \  \beta^*_{\mathrm{Kr}} = p_{x} \varphi{\left(\mathbf{r} \right)}\right)$

    

    So we have the components of a Kropina-type metric on our
Cartan/co-Finsler space, where \begin{equation}
    \alpha^*_{\mathrm{Kr}} = \left( a^{ij}(\mathbf{r})\, p_i\, p_j \right)^{\frac{1}{2}}
    ,\quad
    \beta^*_{\mathrm{Kr}} = p_i b^i(\mathbf{r})
\end{equation}

The Riemannian metric component \(a^{ij}\) is just a Euclidean metric
\begin{equation}
    a^{ij}(\mathbf{r}) 
    =
    \sqrt{\varphi(\mathbf{r})}
    \begin{bmatrix}
        1 & 0 \\
        0 & 1
    \end{bmatrix}
\end{equation}

and the vector field component \(b^i\) (the dual of what would be a
1-form for a Kropina-type metric on a Finsler space) is \begin{equation}
    b^{i}(\mathbf{r}) 
    =
    {\varphi(\mathbf{r})}
    \begin{bmatrix}
        1 \\
        0
    \end{bmatrix}
\end{equation}

 
            
    
    $\displaystyle \left(\alpha^*_{\mathrm{Kr}}\right)^{1 - \eta} \left(\beta^*_{\mathrm{Kr}}\right)^{\eta} = p_{x}^{\eta} \left(p_{x}^{2} + p_{z}^{2}\right)^{\frac{1}{2} - \frac{\eta}{2}} \varphi{\left(\mathbf{r} \right)}$

    

 
            
    
    $\displaystyle \left( \mathcal{F}_* = p_{x}^{\eta} \left(p_{x}^{2} + p_{z}^{2}\right)^{\frac{1}{2} - \frac{\eta}{2}} \varphi{\left(\mathbf{r} \right)}, \  \text{True}\right)$

    

    Therefore \(F^*\) is a generalized \(m\)-Kropina metric, which has the
standard form: \begin{equation}
    F^* = \alpha^{m+1} \beta^{-m}
\end{equation} where \(m:=-\eta\), with the constraint that
\(m \neq 0, -1\), and where \(\alpha := \alpha^*_\mathrm{Kr}\) and
\(\beta:=\beta^*_\mathrm{Kr}\) (don't confuse these \(\alpha\),
\(\beta\) with the ray and surface tilt angles).

For \(\eta=\tfrac{1}{2}\), we have a generalized
\(\left(-\tfrac{1}{2}\right)\)-Kropina metric function
\(F^* = \alpha^{1/2} \beta^{1/2}\).

For \(\eta=\tfrac{3}{2}\), we have a generalized
\(\left(-\tfrac{3}{2}\right)\)-Kropina metric function
\(F^* = \alpha^{-1/2} \beta^{3/2}\).

For \(\eta=1\), the metric function is ill-defined.

Note: the canonical Kropina metric is usually used to describe \(F\) not
\(F^*\), but not exclusively.
